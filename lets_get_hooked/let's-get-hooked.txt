What are React Hooks?
    - React Hooks are javascript functions which have the ability to update the state which cause the re-render on the DOM

Why do we need a useState Hook?
    - useState hook is used to store (or) maintain the state of the component
    - when the state change cause by the setstate it rerenders the component
